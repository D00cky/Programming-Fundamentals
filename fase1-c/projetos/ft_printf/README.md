# ft_printf

> Fase 1 — C

Implementar `printf` com variadic functions e formatadores.

## Critérios de conclusão

- [ ] Suporta conversões: `%c` `%s` `%p` `%d` `%i` `%u` `%x` `%X` `%%`
- [ ] Output idêntico ao `printf` da libc para todos os casos
- [ ] Sem memory leaks (`valgrind` limpo)
- [ ] Linka corretamente com `libft`
- [ ] Compila com `gcc -Wall -Wextra -Werror`
