# libft

> Fase 1 — C

Reimplementar a libc do zero: funções de string, memória, lista ligada.

## Critérios de conclusão

- [ ] Todas as funções obrigatórias implementadas (Part 1 + Part 2)
- [ ] Makefile gera `libft.a` com targets `all`, `clean`, `fclean`, `re`
- [ ] `valgrind --leak-check=full` reporta 0 leaks
- [ ] Tester público (libft-unit-test ou Tripouille) passa sem erros
- [ ] Compila com `gcc -Wall -Wextra -Werror`
