# push_swap

> Fase 1 — C

Ordenação com duas pilhas (`a` e `b`) usando o menor número possível de operações.

## Critérios de conclusão

- [ ] Ordena qualquer lista de inteiros sem repetição
- [ ] 5 números: ≤ 12 operações
- [ ] 100 números: ≤ 700 operações (nota máxima)
- [ ] 500 números: ≤ 5500 operações (nota máxima)
- [ ] Checker (bonus) valida a sequência de operações corretamente
