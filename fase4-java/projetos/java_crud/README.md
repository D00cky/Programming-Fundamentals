# Java CRUD

> Fase 4 — Java

API REST com Spring Boot, JPA/Hibernate e banco de dados relacional.

## Critérios de conclusão

- [ ] Endpoints `GET`, `POST`, `PUT`, `DELETE` para pelo menos um recurso
- [ ] Arquitetura em camadas: `Controller` → `Service` → `Repository`
- [ ] Testes de integração com banco em memória (H2) usando `@SpringBootTest`
- [ ] Pelo menos 3 design patterns identificados e aplicados (ex.: Repository, Builder, Strategy)
- [ ] Documentação da API com Swagger/OpenAPI 3
