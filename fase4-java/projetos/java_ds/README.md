# Java DS

> Fase 4 — Java

Reimplementar estruturas de dados clássicas em Java com generics.

## Critérios de conclusão

- [ ] `ArrayList`, `LinkedList`, `Stack` e `Queue` genéricos implementados do zero
- [ ] `HashMap` com encadeamento separado (separate chaining) e rehashing
- [ ] Suíte JUnit 5 cobre todos os métodos públicos
- [ ] Complexidade de cada operação documentada em Javadoc (`O(n)`, `O(1)` amortizado…)
- [ ] Cobertura de código ≥ 80% medida pelo JaCoCo

## Recursos

- [Algorithms, 4th ed. — Sedgewick & Wayne](https://algs4.cs.princeton.edu/home/)
- [JUnit 5 User Guide](https://junit.org/junit5/docs/current/user-guide/)
- [JaCoCo](https://www.jacoco.org/jacoco/)
