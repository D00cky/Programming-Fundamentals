# Fase 6 — Engenharia de Software

> *"Any fool can write code that a computer can understand. Good programmers write code that humans can understand."* — Martin Fowler

Essa fase não é sobre algoritmos ou linguagens — é sobre como escrever software que funciona em produção, que equipes conseguem manter, e que evolui sem virar uma bola de lama.

---

## Por que essa fase existe?

Programadores que chegam aqui sabem codar. A diferença entre um programador e um engenheiro de software é que o engenheiro pensa em **manutenibilidade**, **testabilidade**, **escalabilidade** e **comunicação** — não só em "funciona".
