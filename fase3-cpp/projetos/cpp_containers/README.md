# CPP Containers

> Fase 3 — C++

Reimplementar `vector`, `map`, `stack` e `set` da STL com iterators completos.

## Critérios de conclusão

- [ ] `ft::vector` com iterators (random access), `reserve`, `resize`, `push_back`
- [ ] `ft::map` com árvore rubro-negra, iterators bidirecionais e `lower_bound`/`upper_bound`
- [ ] `ft::stack` e `ft::set` funcionais e conformes à STL
- [ ] Tester oficial (`mli42/ft_containers-terminator` ou equivalente) passa
- [ ] Compila com `-std=c++98`
